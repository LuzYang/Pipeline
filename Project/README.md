# Book Management System

A simple library management system implemented in C, featuring core CRUD (Create, Read, Update, Delete) operations for book information management.

## 📚 Project Overview

This is a straightforward book management system designed to practice and demonstrate fundamental C programming concepts and syntax. The system allows users to manage a collection of books with basic operations.

## ✨ Features

The system supports the following operations:

- **Create** (Add) - Add new books to the library
- **Read** (Query) - View and search for book information
- **Update** (Modify) - Edit existing book details
- **Delete** (Remove) - Remove books from the library

## 🛠️ Technologies

- **Language**: C
- **Compiler**: GCC
- **Dependencies**: None (standard C library only)

## 🚀 Compilation & Execution

### Simple Compilation

Compile the project using GCC:

```bash
gcc -o book_management main.c
```

Or if you have multiple source files:

```bash
gcc -o book_management src/*.c
```

### Running the Program

Execute the compiled program:

```bash
./book_management
```

## 💻 Usage Example

```bash
# Compile
gcc -o book_management main.c

# Run
./book_management

# Follow the on-screen menu to manage books
```

# Pipeline

A personal repository containing multiple independent projects.

## 📂 Repository Structure

This repository hosts two separate projects that are independent of each other:

### [Ml/](./Ml/)
Machine Learning projects and experiments. This directory contains ML-related work including models, datasets, and notebooks.

### [Project/](./Project/)
C programming projects. This directory contains C-based projects and implementations.

## 🚀 Quick Start

To get started with this repository:

```bash
# Clone the repository
git clone https://github.com/LuzYang/Pipeline.git
cd Pipeline
```

## 📖 Project Details

Each subdirectory has its own README file with detailed documentation:

- **ML Projects**: See [Ml/README.md](./Ml/README.md) for ML-specific setup, requirements, and usage instructions.
- **C Projects**: See [Project/README.md](./Project/README.md) for C-specific setup, build instructions, and usage.
